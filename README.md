# Thomas Remodeling & Property Maintenance Website

## Deploy to Vercel
1. Create a GitHub account.
2. Create a new repository, for example: `thomas-remodeling-site`.
3. Upload all files from this folder to the repository.
4. Go to Vercel.com and sign in with GitHub.
5. Click Add New > Project.
6. Import the GitHub repo.
7. Vercel should auto-detect Vite.
8. Build command: `npm run build`
9. Output directory: `dist`
10. Click Deploy.
